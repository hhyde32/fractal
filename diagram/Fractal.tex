For the following plots I used the polynomial
\[
P(z) = (1+i) + (2+2i)z + (3+3i)z^2 + (4+4i)z^3 + (5+5i)z^4 + (6+6i)z^5
\]
where the origin is at $-2+i$ and the square has a width of $2$.
